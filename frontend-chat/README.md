# frontend-chat · Orbio × Grupo Sazón

App **Next.js** del **canal de chat** para el screening de candidatos a **repartidor/a** (cadena Grupo Sazón, España y México). El candidato habla con el asistente; el backend recoge datos estructurados del proceso de selección.

- **Stack:** Next.js 16, React 19, CSS modules.
- **Copy / marca:** `src/constants/branding.ts` (títulos, bienvenida, placeholders).
- **Relación con el monorepo:** complementa el **frontend-dashboard** (recruiters) y el API de screening en `backend/`.

## Desarrollo

```bash
npm install
npm run dev
```

Abre [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build && npm start
```
